# Veloce Research OS Showcase

This folder is a public-safe static preview page for Veloce Research OS.

It is designed for Vercel, Netlify, or any static host:

```bash
cd showcase
python3 -m http.server 8080
```

Deploy as a preview:

```bash
vercel deploy showcase -y
```

or drag the `showcase` folder into Netlify Drop.

The page intentionally does not expose private OpenWebUI, MCPO, Hermes, or Paperclip URLs.
