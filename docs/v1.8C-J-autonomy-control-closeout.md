## Production Framing

The system is now production-level as a governed autonomy architecture and dry-run control plane.

Live unattended production writes should still graduate through an operator-approved pilot:

```text
1. Run make stack-health.
2. Run make rollback-drill.
3. Run make autonomy-controls.
4. Run make autonomy-readiness.
5. Enable one scoped Paperclip token.
6. Run one comment-only unattended issue closeout.
7. Run one PR-only low-risk canary.
8. Keep human approval for merge/deploy until the pilot is reviewed.
```

## Safety Boundary

The package still forbids:

```text
arbitrary shell
raw Docker control
raw filesystem writes
raw GitHub token access
secret reads
Ruflo runtime execution
browser extension production writes
```
