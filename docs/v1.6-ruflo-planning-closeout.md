## Lessons

- Paperclip is useful as a ledger, but its runner may not have the runtime dependencies needed for validation.
- External validation is acceptable when it is posted back to the issue with exact commands and results.
- Python tests inside Paperclip-generated workspaces may need `PYTHONPATH=.` because the workspace is not packaged.
- Ruflo has a broad surface area. Its value is orchestration, but the safe first use is planning-only output.
- MCPO remains the right bridge for Open WebUI because it exposes explicit, inspectable HTTP tools instead of broad shell access.
